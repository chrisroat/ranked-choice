Use RCV Tabulator to run a ranked choice election to determine the winner, understand round-by-round vote counts, and visualize the vote flow in a Sankey diagram.

To get started, use a Google Form to run your election. For each ranked-choice race, create a question using the following settings.

  • Select the "Multiple Choice Grid" question type
  • Label the rows with each candidate's name
  • Label the columns "1st", "2nd", "3rd", ... [up to the number of rankings you allow]
  • Turn on the "Limit one response per column" setting.
  • Do NOT use the "Require a response in each row" setting, unless it is something you *really* want. In most cases, it is acceptable to allow voters not to rank all candidates.
  • We recommend using the "Shuffle row order" to remove any bias toward candidates listed near the start of the order.

Once voting is complete for your entire election, go to the "Responses" tab and select the "Create Spreadsheet" icon (green square with stripes). It should have the name "Your Form's Name (Responses)".

Then use one of the following methods to create your results:

  • Download the Google Sheet as a csv file, and then upload to analyze.
  • Directly analyze a Google Sheet, but giving the website permission to access your Google Drive.
